# Sysinspector
SysInspector is a professional Windows desktop utility written in Python. 

It gives IT administrators and support staff quick access to important local machine details such as BIOS version, computer system data, CPU, disks, network adapters, operating system information, services, startup entries, and more. 

The application includes a summary dashboard, structured navigation, filtering, clipboard copy, refresh, and report export to make troubleshooting and inventory work easier.
